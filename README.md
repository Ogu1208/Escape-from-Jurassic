# 🦕 Escape-from-Jurassic
평소 공룡을 너무 좋아해 오늘도 어김 없이 공룡 게임을 즐기고 있었어..
엇 근데 지지직.. 소리와 함께 비디오 게임이 잠깐 멈추더니 빨려 들어가 버렸어!

분명 게임 내용은 공룡과 슬라임들을 다 죽이고 잼을 모아야 탈출할수 있었는데.. 혹시 이것도,,?

잘못하면 죽어서 못돌아올 수도 있는거 아니야,,?

우선 파란색 슬라임은 공격하니까 조심하자!!
